# 🎓 CampusPilot AI

## AI-Powered Decision Intelligence Platform for Student Success

CampusPilot AI is an AI-powered Decision Intelligence Platform designed to help engineering students make informed academic and career decisions while enabling faculty to identify common skill gaps and plan targeted training programs.

The platform analyzes a student's academic profile, career aspirations, current skills, and project experience to generate personalized recommendations, placement readiness insights, learning roadmaps, and career guidance. It also provides faculty with analytics to better understand student trends and improve learning outcomes.

---

# 🚀 Problem Statement

Engineering students frequently struggle with questions such as:

- What should I learn next?
- Which skills are required for my dream career?
- Am I placement-ready?
- Which project should I build?
- Which certification should I pursue?

Faculty members also face challenges in identifying common skill gaps across students and planning effective training programs.

CampusPilot AI addresses these challenges by transforming scattered information into structured, AI-powered decision support.

---

# 💡 Solution

CampusPilot AI acts as a personalized AI Decision Intelligence Platform.

Instead of providing generic advice, it analyzes a student's profile and recommends:

- Placement Readiness Score
- Skill Gap Analysis
- Personalized Learning Roadmap
- Recommended Projects
- Certifications
- Career Guidance
- Learning Platforms

Faculty members can also visualize student trends through an analytics dashboard and identify common skill gaps for better intervention planning.

# ▶️ Installation

Clone the repository

```bash
git clone https://github.com/manishkumar0105/CampusPilot-AI.git
```

Navigate to the project directory

```bash
cd CampusPilotAI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
